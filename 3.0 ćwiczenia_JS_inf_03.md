## CZĘŚĆ 1: WYSZUKIWANIE ELEMENTÓW DOM

### Trzy metody wyszukiwania:

| Metoda | Co zwraca? | Kiedy używać? |
|--------|------------|---------------|
| `getElementById()` | Jeden element | Gdy znasz ID elementu |
| `querySelector()` | Jeden element (pierwszy pasujący) | Gdy szukasz po klasie, tagu lub selektorze CSS |
| `querySelectorAll()` | Listę elementów (NodeList) | Gdy chcesz znaleźć WSZYSTKIE pasujące |

---

### getElementById()

Znajduje element po jego **ID** (atrybut `id` w HTML).

```html
<p id="demo1">Jakiś tekst</p>
```

```javascript
const element = document.getElementById('demo1');
element.style.color = 'red';
```

**Uwaga:** ID musi być unikalne na stronie! Bez znaku `#` przed nazwą.

---

### querySelector()

Znajduje **PIERWSZY** element pasujący do selektora CSS.

```javascript
// Po ID (z #)
const element = document.querySelector('#demo1');

// Po klasie (z .)
const element = document.querySelector('.special');

// Po tagu (bez żadnego znaku)
const element = document.querySelector('span');

// Po atrybucie
const element = document.querySelector('[type="text"]');

// Kombinacja: tag z klasą
const element = document.querySelector('p.special');

// Potomek: element wewnątrz innego
const element = document.querySelector('.container p');
```

**Tabela selektorów:**

| Selektor | Składnia | Przykład |
|----------|----------|----------|
| ID | `#nazwaId` | `querySelector('#menu')` |
| Klasa | `.nazwaKlasy` | `querySelector('.active')` |
| Tag | `nazwaTagу` | `querySelector('p')` |
| Atrybut | `[atrybut="wartość"]` | `querySelector('[type="text"]')` |
| Potomek | `rodzic potomek` | `querySelector('div p')` |

---

### querySelectorAll()

Znajduje **WSZYSTKIE** elementy pasujące do selektora. Zwraca **NodeList** (listę).

```html
<p class="item">Element 1</p>
<p class="item">Element 2</p>
<p class="item">Element 3</p>
```

```javascript
const elementy = document.querySelectorAll('.item');
// elementy = [p.item, p.item, p.item] - lista 3 elementów
```

**UWAGA:** Na liście NIE możesz bezpośrednio zmienić stylu! Musisz użyć **forEach**.

---

### forEach - pętla po elementach

Metoda `forEach` wykonuje funkcję dla **KAŻDEGO** elementu z listy.

**Składnia:**
```javascript
lista.forEach(function(element) {
    // kod wykonywany dla każdego elementu
});
```

**Przykład - zmiana koloru wszystkich elementów:**
```javascript
const elementy = document.querySelectorAll('.item');

elementy.forEach(function(element) {
    element.style.color = 'green';
});
```

**Przykład - zmiana wielu stylów:**
```javascript
const karty = document.querySelectorAll('.card');

karty.forEach(function(karta) {
    karta.style.backgroundColor = 'gray';
    karta.style.color = 'white';
    karta.style.padding = '20px';
});
```

**Przykład - zmiana tekstu:**
```javascript
const etykiety = document.querySelectorAll('.label');

etykiety.forEach(function(etykieta) {
    etykieta.innerHTML = 'ZMIENIONY';
});
```

---

### Porównanie metod:

| Chcesz... | Użyj |
|-----------|------|
| Znaleźć element po ID | `getElementById('id')` |
| Znaleźć PIERWSZY element po klasie | `querySelector('.klasa')` |
| Znaleźć PIERWSZY element po tagu | `querySelector('tag')` |
| Znaleźć WSZYSTKIE elementy po klasie | `querySelectorAll('.klasa')` |
| Znaleźć WSZYSTKIE elementy po tagu | `querySelectorAll('tag')` |
| Zmienić WSZYSTKIE znalezione | `querySelectorAll()` + `forEach` |

---

### ⚠️ Częste błędy przy wyszukiwaniu:

```javascript
// ❌ BŁĄD: Brakuje 'document.'
const el = getElementById('demo1');

// ✔ POPRAWNIE:
const el = document.getElementById('demo1');

// ❌ BŁĄD: # przy getElementById
const el = document.getElementById('#demo1');

// ✔ POPRAWNIE: bez #
const el = document.getElementById('demo1');

// ❌ BŁĄD: Brak . przy klasie w querySelector
const el = document.querySelector('special');

// ✔ POPRAWNIE: z kropką
const el = document.querySelector('.special');

// ❌ BŁĄD: Próba zmiany stylu bezpośrednio na liście
const lista = document.querySelectorAll('.item');
lista.style.color = 'red';  // NIE DZIAŁA!

// ✔ POPRAWNIE: użyj forEach
lista.forEach(function(el) {
    el.style.color = 'red';
});
```

## CZĘŚĆ 2: MANIPULACJA STYLAMI CSS

### Jak zmieniać style CSS przez JavaScript?

```javascript
// Schemat:
element.style.nazwaWłaściwości = 'wartość';
```

**UWAGA:** Nazwy właściwości CSS w JavaScript piszemy w **camelCase** (bez myślników)!

| CSS | JavaScript |
|-----|------------|
| `background-color` | `backgroundColor` |
| `font-size` | `fontSize` |
| `font-weight` | `fontWeight` |
| `border-radius` | `borderRadius` |
| `margin-top` | `marginTop` |

---

### Tabela najczęściej używanych właściwości:

#### Kolory i tło

| Właściwość | Składnia JS | Przykład |
|------------|-------------|----------|
| Kolor tekstu | `element.style.color` | `element.style.color = 'red';` |
| Kolor tła | `element.style.backgroundColor` | `element.style.backgroundColor = 'yellow';` |
| Kolor tła (HEX) | `element.style.backgroundColor` | `element.style.backgroundColor = '#ff5733';` |
| Kolor tła (RGB) | `element.style.backgroundColor` | `element.style.backgroundColor = 'rgb(255, 87, 51)';` |

#### Czcionka

| Właściwość | Składnia JS | Przykład |
|------------|-------------|----------|
| Rozmiar czcionki | `element.style.fontSize` | `element.style.fontSize = '20px';` |
| Grubość czcionki | `element.style.fontWeight` | `element.style.fontWeight = 'bold';` |
| Rodzina czcionki | `element.style.fontFamily` | `element.style.fontFamily = 'Arial';` |
| Styl czcionki | `element.style.fontStyle` | `element.style.fontStyle = 'italic';` |
| Wyrównanie tekstu | `element.style.textAlign` | `element.style.textAlign = 'center';` |
| Dekoracja tekstu | `element.style.textDecoration` | `element.style.textDecoration = 'underline';` |

#### Marginesy i padding

| Właściwość | Składnia JS | Przykład |
|------------|-------------|----------|
| Margines (wszystkie) | `element.style.margin` | `element.style.margin = '10px';` |
| Margines górny | `element.style.marginTop` | `element.style.marginTop = '20px';` |
| Margines dolny | `element.style.marginBottom` | `element.style.marginBottom = '20px';` |
| Padding (wszystkie) | `element.style.padding` | `element.style.padding = '15px';` |
| Padding lewy | `element.style.paddingLeft` | `element.style.paddingLeft = '10px';` |

#### Ramki

| Właściwość | Składnia JS | Przykład |
|------------|-------------|----------|
| Ramka | `element.style.border` | `element.style.border = '2px solid black';` |
| Kolor ramki | `element.style.borderColor` | `element.style.borderColor = 'red';` |
| Zaokrąglenie | `element.style.borderRadius` | `element.style.borderRadius = '10px';` |

#### Wymiary i wyświetlanie

| Właściwość | Składnia JS | Przykład |
|------------|-------------|----------|
| Szerokość | `element.style.width` | `element.style.width = '200px';` |
| Wysokość | `element.style.height` | `element.style.height = '100px';` |
| Wyświetlanie | `element.style.display` | `element.style.display = 'none';` |
| Widoczność | `element.style.visibility` | `element.style.visibility = 'hidden';` |

---

### Najczęstsze wartości:

#### Kolory (color, backgroundColor)
```javascript
'red', 'blue', 'green', 'yellow', 'white', 'black', 'gray'
'#ff0000', '#00ff00', '#0000ff'   // HEX
'rgb(255, 0, 0)'                   // RGB
```

#### Grubość czcionki (fontWeight)
```javascript
'normal'    // zwykła
'bold'      // pogrubiona
```

#### Rozmiar czcionki (fontSize)
```javascript
'12px', '16px', '20px', '24px'   // piksele
```

## CZĘŚĆ 3: ZMIANA TREŚCI I ATRYBUTÓW

### Zmiana treści elementu:

| Właściwość | Co robi? | Przykład |
|------------|----------|----------|
| `innerHTML` | Zmienia zawartość HTML | `element.innerHTML = '<b>Tekst</b>';` |
| `innerText` | Zmienia tylko tekst | `element.innerText = 'Tekst';` |

### Zmiana atrybutów HTML:

| Metoda | Co robi? | Przykład |
|--------|----------|----------|
| `setAttribute()` | Ustawia atrybut | `element.setAttribute('href', 'https://google.com');` |
| `getAttribute()` | Pobiera atrybut | `const link = element.getAttribute('href');` |

## 🎯 SCHEMAT PRACY

```javascript
// JEDEN ELEMENT:
// 1. Znajdź element
const element = document.getElementById('mojeId');
// lub: document.querySelector('.mojaKlasa');

// 2. Zmień styl / treść / atrybut
element.style.color = 'red';
element.innerHTML = 'Nowy tekst';
element.setAttribute('href', 'https://...');


// WIELE ELEMENTÓW:
// 1. Znajdź wszystkie elementy
const elementy = document.querySelectorAll('.mojaKlasa');

// 2. Użyj forEach
elementy.forEach(function(element) {
    element.style.color = 'red';
    element.innerHTML = 'Nowy tekst';
});

```
