# Kurs Podstaw JavaScript - Lekcje 3-4

## Lekcja 3: Operatory i działania na zmiennych

### Operatory arytmetyczne

JavaScript oferuje pełen zestaw operatorów matematycznych:

```javascript
let a = 10;
let b = 3;

// Dodawanie
console.log(a + b); // 13

// Odejmowanie
console.log(a - b); // 7

// Mnożenie
console.log(a * b); // 30

// Dzielenie
console.log(a / b); // 3.3333...

// Modulo (reszta z dzielenia)
console.log(a % b); // 1

// Potęgowanie (ES6)
console.log(a ** 2); // 100
```

### Operatory przypisania

```javascript
let x = 5;

x = x + 3;  // x = 8
x += 3;     // To samo co x = x + 3

x -= 2;     // x = x - 2
x *= 2;     // x = x * 2
x /= 2;     // x = x / 2
x %= 3;     // x = x % 3

// Inkrementacja i dekrementacja
let licznik = 0;
licznik++;  // licznik = licznik + 1
licznik--;  // licznik = licznik - 1

// Pre i post inkrementacja
let y = 5;
console.log(y++); // 5 (wyświetla, potem zwiększa)
console.log(y);   // 6

let z = 5;
console.log(++z); // 6 (zwiększa, potem wyświetla)
console.log(z);   // 6
```

### Operatory porównania

```javascript
let a = 5;
let b = 10;
let c = "5";

// Równość (z konwersją typów)
console.log(a == c);   // true (5 == "5")

// Ścisła równość (bez konwersji)
console.log(a === c);  // false (5 !== "5")

// Nierówność
console.log(a != b);   // true
console.log(a !== c);  // true

// Większe, mniejsze
console.log(b > a);    // true
console.log(a < b);    // true
console.log(a >= 5);   // true
console.log(b <= 10);  // true
```

**Ważne:** Zawsze używaj `===` i `!==` zamiast `==` i `!=`!

### Operatory logiczne

```javascript
let x = true;
let y = false;

// AND (&&) - oba warunki muszą być prawdziwe
console.log(x && y);  // false
console.log(true && true);  // true

// OR (||) - przynajmniej jeden warunek musi być prawdziwy
console.log(x || y);  // true
console.log(false || false);  // false

// NOT (!) - negacja
console.log(!x);  // false
console.log(!y);  // true

// Przykłady praktyczne
let wiek = 20;
let maLicencje = true;

console.log(wiek >= 18 && maLicencje);  // true
console.log(wiek < 18 || !maLicencje);  // false
```

### Pierwszeństwo operatorów

```javascript
let wynik1 = 2 + 3 * 4;    // 14 (mnożenie przed dodawaniem)
let wynik2 = (2 + 3) * 4;  // 20 (nawiasy zmieniają kolejność)

// Kolejność: !, **, *, /, %, +, -, porównania, &&, ||
let skomplikowane = 5 + 3 * 2 ** 2 - 1;
console.log(skomplikowane); // 16
// Krok po kroku: 2**2 = 4, 3*4 = 12, 5+12 = 17, 17-1 = 16
```

## Lekcja 4: Działania na stringach (tekstach)

### Właściwości stringów

```javascript
let tekst = "Hello World";

// Długość stringa
console.log(tekst.length); // 11

// Dostęp do znaków (indeksowanie od 0)
console.log(tekst[0]);  // "H"
console.log(tekst[6]);  // "W"
console.log(tekst[tekst.length - 1]); // "d" (ostatni znak)
```

### Łączenie stringów

```javascript
// Operator +
let imie = "Jan";
let nazwisko = "Kowalski";
let pelneImie = imie + " " + nazwisko;
console.log(pelneImie); // "Jan Kowalski"

// Operator +=
let wiadomosc = "Witaj";
wiadomosc += " w ";
wiadomosc += "JavaScript!";
console.log(wiadomosc); // "Witaj w JavaScript!"

// Template literals (najlepsza metoda)
let miasto = "Warszawa";
let kraj = "Polska";
let adres = `Mieszkam w ${miasto}, ${kraj}`;
console.log(adres); // "Mieszkam w Warszawa, Polska"
```

### Metody stringów

#### 1. Zmiana wielkości liter
```javascript
let tekst = "JavaScript jest super!";

console.log(tekst.toUpperCase()); // "JAVASCRIPT JEST SUPER!"
console.log(tekst.toLowerCase()); // "javascript jest super!"
```

#### 2. Wyszukiwanie w stringach
```javascript
let zdanie = "Uczę się JavaScript";

// indexOf - zwraca pozycję pierwszego wystąpienia
console.log(zdanie.indexOf("się"));        // 6
console.log(zdanie.indexOf("Python"));     // -1 (nie znaleziono)

// includes - sprawdza czy string zawiera podstring
console.log(zdanie.includes("Java"));      // true
console.log(zdanie.includes("Python"));    // false

// startsWith i endsWith
console.log(zdanie.startsWith("Uczę"));    // true
console.log(zdanie.endsWith("Script"));    // true
```

#### 3. Wycinanie fragmentów
```javascript
let tekst = "JavaScript";

// slice(start, end)
console.log(tekst.slice(0, 4));   // "Java"
console.log(tekst.slice(4));      // "Script"
console.log(tekst.slice(-6));     // "Script" (od końca)

// substring(start, end) - podobne do slice
console.log(tekst.substring(0, 4)); // "Java"

// substr(start, length) - PRZESTARZAŁE, unikaj!
```

#### 4. Zastępowanie tekstu
```javascript
let tekst = "Lubię Pythona";

// replace - zastępuje pierwsze wystąpienie
let nowy = tekst.replace("Pythona", "JavaScript");
console.log(nowy); // "Lubię JavaScript"

// replaceAll - zastępuje wszystkie wystąpienia (ES2021)
let powtorzenia = "kot kot kot";
console.log(powtorzenia.replaceAll("kot", "pies")); // "pies pies pies"
```

#### 5. Dzielenie i łączenie
```javascript
// split - dzieli string na tablicę
let zdanie = "JavaScript jest super";
let slowa = zdanie.split(" ");
console.log(slowa); // ["JavaScript", "jest", "super"]

let data = "2024-01-15";
let czesciDaty = data.split("-");
console.log(czesciDaty); // ["2024", "01", "15"]

// Łączenie znaków
let litery = "JavaScript".split("");
console.log(litery); // ["J", "a", "v", "a", "S", "c", "r", "i", "p", "t"]
```

#### 6. Usuwanie białych znaków
```javascript
let tekst = "   JavaScript   ";

console.log(tekst.trim());      // "JavaScript"
console.log(tekst.trimStart()); // "JavaScript   "
console.log(tekst.trimEnd());   // "   JavaScript"
```

#### 7. Powtarzanie stringa
```javascript
let znak = "*";
console.log(znak.repeat(5)); // "*****"

let spacja = " ";
console.log("Tab" + spacja.repeat(10) + "Content");
```

### Escape sequences (sekwencje specjalne)

```javascript
// Nowa linia
console.log("Linia 1\nLinia 2");

// Tabulator

// Cudzysłów wewnątrz stringa

console.log('To\'s super!');

// Backslash
console.log("Ścieżka: C:\\Users\\Desktop");
```

### Najważniejsze zasady:
- Zawsze używaj `===` zamiast `==` dla porównań
- Template literals (`` ` ` ``) to najlepszy sposób łączenia stringów
- Stringi w JavaScript są **immutable** (niezmienne)
- Metody stringów zwracają **nowy** string, nie modyfikują oryginału
