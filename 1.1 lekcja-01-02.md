## Lekcja 1: Wprowadzenie do JavaScript i pierwsze kroki

### Czym jest JavaScript?

JavaScript to język programowania, który pozwala na tworzenie interaktywnych stron internetowych. Został stworzony w 1995 roku i dziś jest jednym z najważniejszych języków programowania w świecie web developmentu.

**Gdzie używamy JavaScript?**
- Manipulacja treścią strony (DOM)
- Reagowanie na działania użytkownika
- Tworzenie animacji
- Komunikacja z serwerami
- Tworzenie aplikacji webowych

### Jak uruchomić JavaScript?

JavaScript możemy uruchamiać na kilka sposobów:

1. **W konsoli przeglądarki** (F12 lub Ctrl+Shift+I)
2. **W pliku HTML** za pomocą tagu `<script>`
3. **W zewnętrznym pliku .js**

#### Przykład 1: Inline w HTML
```html
<!DOCTYPE html>
<html>
<head>
    <title>Mój pierwszy JavaScript</title>
</head>
<body>
    <h1>Witaj w JavaScript!</h1>
    
    <script>
        console.log("Hello, World!");
    </script>
</body>
</html>
```

#### Przykład 2: Zewnętrzny plik
```html
<!DOCTYPE html>
<html>
<head>
    <title>Mój pierwszy JavaScript</title>
    <script src="script.js"></script>
</head>
<body>
    <h1>Witaj w JavaScript!</h1>
</body>
</html>
```

Plik `script.js`:
```javascript
console.log("Hello, World!");
```

## Lekcja 2: Zmienne i typy danych

### Czym są zmienne?

Zmienne to "pojemniki" na dane. Pozwalają nam przechowywać i używać wartości w programie.

### Deklaracja zmiennych: var, let, const

JavaScript oferuje trzy sposoby deklarowania zmiennych:

#### 1. **let** - dla zmiennych, które mogą się zmieniać
```javascript
let imie = "Anna";
let wiek = 25;

console.log(imie); // Anna
console.log(wiek); // 25

// Możemy zmienić wartość
wiek = 26;
console.log(wiek); // 26
```

#### 2. **const** - dla stałych (nie można zmienić wartości)
```javascript
const PI = 3.14159;
const KRAJ = "Polska";

console.log(PI); // 3.14159

// PI = 3.14; // BŁĄD! Nie można zmienić const
```

#### 3. **var** - stary sposób (unikaj!)
```javascript
var stareZmienne = "Lepiej używaj let lub const";
```

**Zasada:** Używaj `const` domyślnie. Jeśli wartość musi się zmieniać, użyj `let`. Unikaj `var`.

### Podstawowe typy danych

#### 1. Number (Liczby)
```javascript
let calkowita = 42;
let zmiennoprzecinkowa = 3.14;
let ujemna = -17;

console.log(typeof calkowita); // "number"
```

#### 2. String (Tekst)
```javascript
let tekst1 = "Hello";
let tekst2 = 'World';
let tekst3 = `Template literal`;

// Łączenie stringów (konkatenacja)
let powitanie = tekst1 + " " + tekst2;
console.log(powitanie); // "Hello World"

// Template literals
let imie = "Jan";
let wiek = 30;
let wiadomosc = `Mam na imię ${imie} i mam ${wiek} lat.`;
console.log(wiadomosc); // "Mam na imię Jan i mam 30 lat."
```

#### 3. Boolean (Wartości logiczne)
```javascript
let czyPrawda = true;
let czyFalsz = false;

console.log(typeof czyPrawda); // "boolean"
```

#### 4. Undefined i Null
```javascript
let niezdefiniowane;
console.log(niezdefiniowane); // undefined

let puste = null;
console.log(puste); // null
```

### Operator typeof

```javascript
console.log(typeof 42);        // "number"
console.log(typeof "tekst");   // "string"
console.log(typeof true);      // "boolean"
console.log(typeof undefined); // "undefined"
console.log(typeof null);      // "object" (znany bug!)
```

### Konwersja typów

```javascript
// String na Number
let tekstLiczba = "123";
let liczba = Number(tekstLiczba);
console.log(liczba); // 123

// Number na String
let liczba2 = 456;
let tekst = String(liczba2);
console.log(tekst); // "456"

// Automatyczna konwersja
console.log("5" + 3);  // "53" (konkatenacja)
console.log("5" - 3);  // 2 (odejmowanie)
console.log("5" * "2"); // 10 (mnożenie)
```
