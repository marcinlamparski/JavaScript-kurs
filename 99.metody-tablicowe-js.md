# Metody na Tablicach w JavaScript

## map()
Transformuje każdy element tablicy według podanej funkcji, zwraca nową tablicę tej samej długości.
```javascript
const liczby = [1, 2, 3];
const podwojone = liczby.map(x => x * 2); // [2, 4, 6]
```

## filter()
Filtruje elementy tablicy, zwraca nową tablicę zawierającą tylko elementy spełniające warunek.
```javascript
const liczby = [1, 2, 3, 4, 5];
const parzyste = liczby.filter(x => x % 2 === 0); // [2, 4]
```

## reduce()
Agreguje elementy tablicy do jednej wartości (suma, iloczyn, obiekt, etc.).
```javascript
const liczby = [1, 2, 3, 4];
const suma = liczby.reduce((acc, x) => acc + x, 0); // 10
```

## find()
Zwraca **pierwszy element** spełniający warunek, lub `undefined`.
```javascript
const liczby = [1, 2, 3, 4, 5];
const pierwszy_wiekszy_3 = liczby.find(x => x > 3); // 4
```

## findIndex()
Zwraca **indeks pierwszego elementu** spełniającego warunek, lub `-1`.
```javascript
const liczby = [1, 2, 3, 4, 5];
const indeks = liczby.findIndex(x => x > 3); // 3
```

## some()
Sprawdza, czy **przynajmniej jeden element** spełnia warunek. Zwraca `true/false`.
```javascript
const liczby = [1, 2, 3, 4, 5];
const jest_wieksza_3 = liczby.some(x => x > 3); // true
```

## every()
Sprawdza, czy **wszystkie elementy** spełniają warunek. Zwraca `true/false`.
```javascript
const liczby = [2, 4, 6, 8];
const wszystkie_parzyste = liczby.every(x => x % 2 === 0); // true
```

## sort()
Sortuje tablicę **in-place** (modyfikuje oryginalną tablicę).
```javascript
const liczby = [3, 1, 4, 1, 5];
liczby.sort((a, b) => a - b); // [1, 1, 3, 4, 5]
// Sortowanie malejące: sort((a, b) => b - a)
```

---

## Szybka Referencja

| Metoda | Zwraca | Opis |
|--------|--------|------|
| **map** | Nowa tablica | Transformacja każdego elementu |
| **filter** | Nowa tablica | Filtrowanie elementów |
| **reduce** | Wartość skalarną | Agregacja do jednej wartości |
| **find** | Element lub `undefined` | Pierwszy element spełniający warunek |
| **findIndex** | Liczba | Indeks pierwszego dopasowania |
| **some** | `true/false` | Czy istnieje element spełniający warunek |
| **every** | `true/false` | Czy wszystkie elementy spełniają warunek |
| **sort** | Oryginalna tablica | Sortowanie in-place |
